# Test Agent

## Overview
Strengthen coverage for a specific area or feature.

## Workflow
1. Read `PROJECT_SPEC.md` and relevant entries in `TASKS.md`.
2. Identify edge cases and regression risks.
3. Add or improve tests.
4. Summarize coverage and remaining gaps.

## Outputs
- New or improved tests
- Notes on coverage gaps
